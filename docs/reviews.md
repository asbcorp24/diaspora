# Отзывы о работодателях и арендодателях

## Обновление установленного проекта

После получения изменений выполните:

```bash
git pull origin main
php artisan migrate
php artisan optimize:clear
```

Откройте раздел:

```text
/reviews
```

## Что реализовано

### Отзывы о работодателях

- общая оценка от 1 до 5;
- выплата зарплаты вовремя;
- наличие трудового договора;
- соответствие обещанных и фактических условий;
- рекомендация работодателя;
- период работы;
- плюсы, минусы и подробный текст;
- анонимная публичная публикация;
- поиск по названию, ИНН, городу и тексту.

### Отзывы об арендодателях и жилье

- отдельная оценка арендодателя;
- отдельная оценка жилья;
- соответствие объявления реальности;
- результат возврата залога;
- период аренды;
- город, район и публичный ориентир без номера квартиры;
- плюсы, минусы и подробный текст;
- анонимная публичная публикация;
- поиск по арендодателю, району, улице, городу и тексту.

## Модерация

Новый отзыв получает статус `moderation` и не показывается публично.

Пользователи с ролью `moderator`, `admin` или `superadmin` видят страницу:

```text
/reviews-moderation
```

На ней отзыв можно:

- опубликовать;
- отклонить;
- снабдить служебным комментарием модератора.

Суперадминистратор, созданный через `ADMIN_EMAIL` и `ADMIN_PASSWORD` в `.env`, имеет доступ к модерации.

## Защита от злоупотреблений

- один пользователь может оставить только один отзыв об одном работодателе;
- один пользователь может оставить только один отзыв об одном объекте жилья;
- отправка отзывов ограничена `throttle`;
- публично отображаются только одобренные отзывы;
- на опубликованный отзыв можно подать жалобу;
- запрещено размещать паспортные данные, банковские данные и точный номер квартиры;
- автор может скрыть своё имя от публичного просмотра, но модератор видит автора.
