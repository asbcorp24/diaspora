# Diaspora

Мультидоменная социально-информационная платформа на **Laravel 9**, **PHP 8.1** и **MySQL**.

Один экземпляр приложения обслуживает несколько национальных диаспор. По доменному имени выбираются название сообщества, родной язык, русский язык, оформление и отдельный контент.

## Реализованные разделы

- определение диаспоры по доменному имени;
- русский и родной язык выбранной диаспоры;
- регистрация пользователей 18+ по телефону, email необязателен;
- анкеты, знакомства, общение и личные сообщения;
- публикации сообщества;
- вакансии и модерация работодателей;
- отзывы о работодателях;
- отзывы об арендодателях и жилье;
- жалобы на отзывы и очередь модерации;
- генератор двуязычных писем в учреждения;
- раздел безопасности и обращения к модераторам;
- новости диаспоры;
- полноценная административная панель;
- журнал действий администраторов;
- управление диаспорами и доменами.

## Требования

- PHP 8.0.2–8.1, рекомендуется PHP 8.1;
- Composer 2;
- MySQL 5.7+ или MySQL 8.0;
- расширения PHP: `mbstring`, `openssl`, `pdo_mysql`, `tokenizer`, `xml`, `ctype`, `json`, `bcmath`, `fileinfo`.

## Установка

```bash
git clone https://github.com/asbcorp24/diaspora.git
cd diaspora
composer install
cp .env.example .env
php artisan key:generate
```

Создайте базу данных:

```sql
CREATE DATABASE diaspora CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

Настройте `.env`:

```dotenv
APP_NAME=Diaspora
APP_ENV=local
APP_DEBUG=true
APP_URL=http://diaspora.test

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=diaspora
DB_USERNAME=root
DB_PASSWORD=

ADMIN_EMAIL=admin@example.com
ADMIN_PHONE=+70000000000
ADMIN_PASSWORD=ChangeMe123!
```

Обязательно замените пароль администратора. Затем выполните:

```bash
php artisan migrate --seed
php artisan optimize:clear
php artisan serve
```

Откройте `http://127.0.0.1:8000`.

## Административная панель

Адрес:

```text
/admin
```

Войти можно по email или телефону суперadministratora из `.env`.

### Роли

- `moderator` — вакансии, публикации, отзывы, жалобы, обращения и новости;
- `admin` — дополнительно пользователи, работодатели, арендодатели, письма, материалы безопасности и журнал действий своей диаспоры;
- `superadmin` — дополнительно управление ролями, всеми пользователями, диаспорами и доменами.

### Разделы админки

- обзор и статистика;
- пользователи;
- вакансии;
- публикации;
- отзывы;
- жалобы на отзывы;
- обращения раздела безопасности;
- новости;
- работодатели;
- арендодатели;
- шаблоны писем;
- материалы безопасности;
- диаспоры и домены;
- журнал действий.

## Новости

Публичная лента:

```text
/news
```

Создание и редактирование:

```text
/admin?section=news
```

Новость поддерживает:

- русский и родной язык;
- категорию;
- заголовок;
- краткое описание;
- полный текст;
- ссылку на изображение обложки;
- черновик или публикацию;
- закрепление наверху;
- отдельный URL по slug.

Последние опубликованные новости автоматически выводятся на главной странице.

## Мультидиаспорность

Основные таблицы:

- `diasporas` — название, языки, цветовая тема и активность;
- `diaspora_domains` — домены, привязанные к диаспоре;
- весь пользовательский контент содержит `diaspora_id`;
- администратор работает только с текущей диаспорой;
- суперadministratora может создавать новые диаспоры и домены.

Пример:

- `uz.example.ru` → русский + узбекский;
- `tj.example.ru` → русский + таджикский;
- `kg.example.ru` → русский + кыргызский.

Для локальной разработки добавьте в `hosts`:

```text
127.0.0.1 diaspora.test
127.0.0.1 uz.diaspora.test
```

Сидер создаёт домены `diaspora.test`, `uz.diaspora.test`, `localhost` и `127.0.0.1`.
