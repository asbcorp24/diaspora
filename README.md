# Diaspora

Мультидиаспорная социально-информационная платформа на Laravel 9 и MySQL.

Один код обслуживает несколько доменов. По доменному имени определяется диаспора, её языки, оформление и контент.

## Основные модули

- профили и знакомства 18+;
- публикации, комментарии, контакты и личные сообщения;
- вакансии, работодатели и отклики;
- генератор обращений и писем в учреждения;
- раздел «Безопасность и правовая помощь»;
- жалобы, блокировки, модерация и проверка организаций;
- русский и родной язык выбранной диаспоры;
- управление диаспорами и доменами из одной базы данных.

## Требования

- PHP 8.0.2–8.1;
- Laravel 9;
- MySQL 5.7+ или 8.0;
- Composer 2.

Подробная установка и архитектура будут добавляться вместе с кодом проекта.
